# 如何連接我們的伺服器？

## 警告！

**我們的伺服器有兩種，`最新版程式伺服器`不會一直開啟，`恆開伺服器`則會一直開啟。**

---

## 下載洋蔥瀏覽器

請先至[此處](http://andrewsource.usa.cc/tor/browser/download)或複製下方連結下載並安裝洋蔥瀏覽器。

`https://www.torproject.org/projects/torbrowser.html`
